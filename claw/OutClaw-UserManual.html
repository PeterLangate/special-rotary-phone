Chapter 1 — Introduction

Chapter 1Meet OutClaw

OutClaw is an Outlook add-in from kalmstrom.com Business Solutions. It opens a chat panel beside your inbox, and from that panel you can ask an AI to read, sort, draft and send mail, manage your calendar, contacts and tasks — using plain English instead of menus, dialogs and mouse clicks.

1.1 What it is

OutClaw is a Microsoft Outlook desktop add-in built with VSTO (Visual Studio Tools for Office). It runs inside the Outlook process and drives your live mailbox through the native Outlook object model (via the Redemption library, which avoids the Outlook security prompts that would otherwise interrupt every operation). Every action it takes is a real Outlook operation that you can see in front of you, undo where undo applies, and audit in the Sent or Deleted Items folder afterwards.

Under the hood it talks to Anthropic's Claude language model for conversation and reasoning. You supply your own Anthropic API key — the add-in does not run through any kalmstrom-hosted service.

1.2 What makes it different

Lives inside Outlook

No separate app, no browser tab, no copy and paste. The add-in talks directly to the mailbox you already have open.

Works on real items

Every action is a normal Outlook operation. Drafts land in Drafts, appointments land in your calendar, moves are real folder moves.

One hundred specialist tools

More than one hundred purpose-built tools for mail, calendar, contacts, tasks, rules, search, archive, attachments and folders — Redemption-backed where the Outlook Object Model Guard would otherwise get in the way, never a generic "pretend to be a human clicking" layer.

1.3 Who it is for

  • Professionals with heavy inboxes who want an assistant that can triage, summarise and file at the pace of their actual email volume.

Plain speech works. You do not need to learn command names. "Find all unread mail from Acme about the renewal, summarise the thread, and draft a short reply confirming we will send the numbers on Friday" is a perfectly good instruction.

1.6 Safety model

OutClaw treats any action that cannot be undone — sending mail, deleting items, moving items into Deleted Items, overwriting a contact — as requiring explicit confirmation. When the AI proposes such an action it calls a confirmation tool first, which surfaces a dialog in the chat panel showing you exactly what is about to happen. The operation only proceeds if you approve. Read-only browsing (listing folders, reading messages, searching) never pauses.

Chapter 2Installation & Setup

Before you can use OutClaw you need to install the add-in, get an Anthropic API key, and enter it in the settings window. The whole process takes about five minutes.

2.1 Install the add-in

  1. Close every open Outlook window.
  2. Run the OutClaw installer (OutClaw Setup.exe or the ClickOnce link supplied by kalmstrom.com).
  3. Follow the wizard — accept the prompts to install the Visual Studio Tools for Office (VSTO) runtime if it asks.
  4. Open Outlook. On the Home tab you should see a new OutClaw button.
Classic Outlook check. Before looking for the button, make sure you are running classic Outlook, not new Outlook for Windows. If a Try the new Outlook toggle is visible in the top-right corner of the window, leave it Off. If the toggle reads New Outlook, click it once to switch back to classic — the add-in only loads in the classic client. The window title bar of classic Outlook says Outlook and shows the familiar ribbon across the top.
The button is missing? Go to File → Options → Add-ins. At the bottom, change Manage to COM Add-ins, click Go… and make sure OutClaw is ticked. If it is in the Disabled Items list, re-enable it there.
Outlook Redemption is bundled. The installer drops Redemption.dll and Redemption64.dll next to the add-in and loads the correct bitness at runtime — no regsvr32, no elevation required.

2.4 Grant mailbox access

The first time OutClaw touches a message, calendar item, contact or task, Outlook may show a security prompt asking whether to allow programmatic access. OutClaw uses Outlook Redemption, which bypasses most of these prompts through the Safe*Item / RDO APIs. If you still see a prompt:

  • Allow access for a reasonable duration (10 minutes is typical).
  • If your organisation enforces strict anti-virus policy on Outlook, ask your IT to confirm the OutClaw publisher certificate.

3.3 How OutClaw thinks

Every instruction you type goes to Claude together with a short description of your current Outlook context (which folder is selected, what items are visible). Claude breaks the request into one or more tool calls — precise operations like list messages in the Inbox, newest 10 — which OutClaw then runs against Outlook through the Redemption library. After each batch of results, Claude decides whether it is done or needs more.

You do not need to know which tool is called when. But it helps to know that:

  • OutClaw can see your folder structure, message headers and bodies, calendar events, contacts and tasks — read operations are always safe to run.
  • Write operations (compose, send, move, delete, create) are explicit. You have to ask for them by name or intent.
  • Sending mail always requires confirmation. Even if you write "send without asking" the tool will ignore that and ask.
  • It will tell you what it did, including the entry ids of items it created, so you can refer back to them.

3.4 Undo and recovery

Outlook has limited undo. OutClaw actions that are reversible:

  • Move — the item still exists, just in a different folder. Ask OutClaw to move it back.
  • Mark read / unread — trivially reversible.
  • Draft creation — delete the draft from the Drafts folder.
  • Appointment, contact, task creation — delete the item from its folder.
Irreversible actions. Send transmits the mail — recall is limited and unreliable. Delete from Deleted Items is a hard delete. OutClaw asks for confirmation before any of these.

3.5 When to give up and switch tools

OutClaw is good at reading, filing and composing. It is not a substitute for your judgement on tone, timing or whether to respond at all. A common pattern: let OutClaw triage the inbox and draft responses, then take over manually for the last five percent of tone and nuance.

4.4 Composing and sending

Drafting and sending are split into three tools on purpose. You can always inspect a draft before it leaves your outbox.

ToolWhat it does
compose_mailCreates a new draft in the Drafts folder. Does not send. Returns the new entry id.
replyCreates a reply draft to an existing message (reply, reply-all or forward).
send_mailSends an existing draft by entry id. Irreversible — always confirmed.
Reply-all to the latest message in the "Pricing review" thread. Confirm we'll have the revised sheet by Friday, thank them for the patience. Keep it under 60 words.
Draft a new mail to procurement@client.example asking for their preferred invoice format. Cc my manager. Professional but warm tone.
HTML or plain text? If you ask for rich formatting (bold, lists, links), OutClaw uses an HTML body. Otherwise it uses plain text. You can also pass html_body explicitly when you need full control.

4.5 Confirmation flow

Any irreversible action — sending mail, hard-deleting, or other one-way operations — triggers a confirmation prompt. The flow looks like this:

  1. You ask OutClaw to do something — e.g. "send that draft".
  2. OutClaw summarises what it is about to do and asks: Send this draft to these recipients? Yes / No.
  3. You answer. "Yes" sends. Anything else cancels.

This flow is enforced at the tool level, not the model level. Even if you instruct OutClaw to skip confirmation, the confirmation still happens. This is a safety feature, not a bug.

Chapter 5 — Calendar

Chapter 5Calendar

Two tools cover your calendar: one to see what is on it and one to add to it. They read and write the default Exchange calendar your Outlook is configured to use.

5.1 Listing calendar events

list_calendar_events returns appointments and meetings in a date range. The default range is the next seven days.

Show me everything on my calendar this week. Highlight anything that overlaps.
List all meetings longer than one hour in the next month. I want to decide which ones I can shorten.

For each event OutClaw returns:

  • Subject, start and end time (in your local timezone).
  • Location or Teams/online link.
  • Organiser and attendee list (up to a reasonable limit).
  • Show-as state (Free, Tentative, Busy, Out of Office).
  • Whether it is recurring or a single occurrence.

5.2 Creating appointments and meetings

create_appointment books a new item on your calendar. If you pass attendees, it becomes a meeting invitation; if not, it stays as a personal appointment.

ParameterNotes
SubjectRequired.
Start / endRequired. Accepts natural language ("Tuesday 10am for 30 minutes") and ISO timestamps.
AttendeesOptional. Semicolon-separated. Becomes a meeting request when present.
LocationOptional.
BodyOptional. Plain text or HTML.
Show-asOptional. Defaults to Busy.
Book a 45-minute Teams meeting with anna@example.com and bob@example.com on Thursday at 14:00, titled "Q2 planning". Add a short agenda in the body.
Teams links. Teams meeting generation is triggered by the Outlook client — OutClaw sets the right flag on the item, but the link is minted by Outlook itself the moment the invitation is sent. If you see a placeholder and not the final URL, open the item in Outlook and it will fill in.

Chapter 8 — Why classic Outlook

Chapter 8Why classic Outlook

OutClaw runs only in classic Outlook for Windows. This is a deliberate choice, not an oversight. This chapter explains the difference between the two Outlook clients, why OutClaw's tools cannot be built on the web add-in API, and what that means for you as a user.

8.1 Two Outlooks, one name

Microsoft ships two desktop programs that both call themselves "Outlook":

Classic Outlook for Windows

The Win32 desktop program — the one that has shipped with Office since 1997. Runs as OUTLOOK.EXE, uses the native Outlook object model over COM, and caches mail locally in an OST file. This is what OutClaw plugs into.

New Outlook for Windows

A rewrite based on Outlook on the web (OWA), shipped as a Progressive Web App inside a WebView2 container. Introduced in 2022 and now distributed with Windows 11 as Outlook (new). Has no COM surface and does not load VSTO add-ins of any kind.

8.4 The trade-off

Running as a VSTO add-in buys OutClaw depth but costs reach. The concrete trade-offs:

You gainYou lose
Every folder, every account, every item typeMac support

Raw MAPI properties via RedemptionPer-tenant manifest deployment
Send-account, send-on-behalf-of, signaturesAuto-updates from Microsoft's add-in infrastructure

8.5 What if Microsoft forces everyone onto new Outlook?

Microsoft has said classic Outlook will remain available alongside new Outlook for at least several years, and enterprise commitments typically extend that window. Even so, it is fair to ask what happens if new Outlook eventually replaces classic outright.

  • Office.js coverage is widening. Microsoft is adding APIs to the web add-in surface in response to developer pressure — folder iteration, rule management, category access, send-on-behalf. It is not a straight replacement today, and probably not next year, but the gap is narrowing.
  • A future Office.js OutClaw would be a different product. Some current tools have no equivalent (rule management, offline reads, send-account selection). Others could be rebuilt on top of Office.js plus Microsoft Graph calls.
  • No sunset is planned. kalmstrom.com will continue to ship OutClaw for classic Outlook for as long as Microsoft supports classic Outlook.

8.6 Practical checklist

  1. If you see a Try the new Outlook toggle in the top-right corner of your window, keep it Off.
  2. If Windows Start lists two Outlooks (Outlook and Outlook (new)), launch the one called simply Outlook.
  3. If your organisation has deployed new Outlook by policy, ask IT whether you can opt back into classic — the classic client is still licensed for the same Microsoft 365 subscription.
  4. If you are stuck on new Outlook, OutClaw cannot help you today. Watch this manual — a future edition may document an Office.js companion for the Office.js-compatible subset of features.

Chapter 9 — Settings

Chapter 9Settings

OutClaw's settings window is reached by clicking the gear icon in the chat panel header. This chapter explains every field.

9.1 API key

Your Anthropic API key. Stored encrypted with Windows DPAPI under %APPDATA%\OutClaw\. The key is redacted in the field — click Show to reveal it temporarily.

Rotate your key if you share a machine. DPAPI protects keys from other Windows accounts on the same machine, but if you allow another person to use your Windows session, they can use your API key. Generate a new key in the Anthropic console and paste it here.

9.2 Model

Chooses which Claude model answers your messages. Recommended defaults:

ModelWhen to use
HaikuFast and cheap. Good for routine tasks with small inputs and short replies.
SonnetBalanced default. Handles summarisation, multi-step reasoning and longer drafts.
OpusHighest quality, slower and more expensive. Use for nuanced tone, sensitive material or hard judgement calls.

9.3 Confirmation policy

Controls how OutClaw treats write operations.

  • Always ask (default) — every write operation requires confirmation.
  • Ask for sends and deletes only — moves, mark-read-unread, compose, create appointment/contact/task run without an extra step. Send and delete always confirm.
  • Ask for sends only — only send requires confirmation. Use this if you trust the agent and want fewer prompts.
Send always confirms. Regardless of this setting, sending mail is always gated behind an explicit yes / no prompt. This is hard-coded and cannot be disabled.

9.4 Logging

Controls the local log file used for diagnostics.

  • Off — no log written.
  • Errors only (default) — only failures are recorded.
  • Verbose — every tool call, input and result is written to the log.

Log file location: %APPDATA%\OutClaw\outclaw.log. It is a plain text file you can open in Notepad. Document, mailbox or workbook contents are not logged at any verbosity level.

9.5 Panel position

By default the chat panel docks on the right of the main Outlook window. You can float it or dock it on the left. The position is remembered per Windows profile.

Chapter 10 — Tips & troubleshooting

Chapter 10Tips & troubleshooting

10.1 Prompt patterns that work

  • Be specific about the scope. "The last 10 unread in Inbox" beats "recent mail".
  • Separate drafting from sending. Ask for a draft, read it, then ask to send. You stay in control and save tokens.
  • Anchor on an item. Once OutClaw has listed a message and cited its entry id, follow-up references like "that last one" or "Anna's message" resolve reliably.
  • Batch your triage. "Summarise all unread in one pass" is cheaper and faster than asking one at a time.

10.2 Common pitfalls

SymptomCauseFix
OutClaw says it can't find the InboxNo default store, or OST not yet synced after a restart.Wait for Outlook to finish syncing; then retry.
Send confirmation fires twiceTwo OutClaw panels open in different Outlook windows.Close the second Outlook window — OutClaw is a single-panel add-in.
Meeting link is blankTeams link is minted when the invitation is sent by Outlook.Open the meeting in Outlook and send. The link populates.
Search returns nothingOutlook search index is not current.Check File → Options → Search → Indexing Options. Rebuild if needed.
Prompts repeatedly ask for permissionOutlook security settings require programmatic access confirmation.Grant for a longer duration, or ask IT to update the Outlook trust settings.

10.3 Cost and token tips

  • Listing is cheap. Summarising long bodies is expensive. Ask for the cheapest step first, then follow up.
  • Haiku is often good enough for inbox triage. Switch to Sonnet for drafts and Opus only when tone really matters.
  • Clearing the conversation periodically resets the context and reduces token usage on subsequent turns.

10.4 Bug-report helpers

Two built-in tools capture the state support needs:

  • get_version_info — ask "what version of OutClaw am I running?" and the reply includes add-in name, auto-refreshed build version, the Outlook version, OS, and .NET runtime. Same version also appears in the top-right corner of the Settings dialog.
  • list_tools — ask "which OutClaw tools mention 'meeting'?" and the agent filters the full tool catalogue by any substring. Useful when you aren't sure whether a capability exists for your scenario.

10.5 Where to get help

  • Log file — %APPDATA%\OutClaw\outclaw.log. Set logging to Verbose before reproducing the issue.
  • kalmstrom.com support — send the log excerpt (with your API key redacted) and a description of what you asked and what happened.
  • Anthropic status — if OutClaw replies with "Claude is unreachable", check status.anthropic.com before worrying about your install.
