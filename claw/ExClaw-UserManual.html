Chapter 1 — Introduction

Chapter 1Meet ExClaw

ExClaw is an Excel add-in from kalmstrom.com Business Solutions. It opens a chat panel on the right side of your workbook, and from that panel you can ask an AI to read, edit, format and reorganise your spreadsheet — ranges and formulas, tables and pivot tables, charts, named ranges, comments, conditional formats and more — using plain English instead of ribbon menus.

1.1 What it is

ExClaw is a Microsoft Excel add-in built with VSTO (Visual Studio Tools for Office). It runs inside the Excel process — not as a separate app — and reads your live workbook through the Excel object model.

Under the hood it talks to Anthropic's Claude language model for conversation and reasoning. You supply your own Anthropic API key — the add-in does not run through any kalmstrom-hosted service.

1.2 What makes it different

Lives inside Excel

No separate app, no browser, no copy and paste. ExClaw reads and drives the same workbook Excel already has open.

Works on the live file

Every action is a normal Excel edit you can undo with Ctrl+Z. No stale snapshots, no external copies.

Ninety-plus specialist tools

Purpose-built tools for ranges, tables, pivot tables, charts, formatting, named ranges, conditional formats, sorting, protection, import/export and more.

  • Analysts who inherit unfamiliar workbooks and need a quick orientation.

Chapter 2 — Installation & Setup

Chapter 2Installation & Setup

Installing ExClaw takes about five minutes: install the add-in, get an Anthropic API key, and paste it into the settings window.

2.1 Install the add-in

  1. Close every open Excel window.
  2. Run the ExClaw installer (ExClaw Setup.exe or the ClickOnce link supplied by kalmstrom.com).
  3. Follow the wizard — accept the prompts to install the Visual Studio Tools for Office (VSTO) runtime if it asks.
  4. Open Excel. On the Home tab you should see a new ExClaw button.
The button is missing? Go to File → Options → Add-ins. At the bottom, change Manage to COM Add-ins, click Go… and make sure ExClaw is ticked. If it is in the Disabled Items list, re-enable it there.

2.2 Get your Anthropic API key

Claude is the language model that powers the conversation. You need your own key so that usage is billed to you, not shared.

  1. Go to console.anthropic.com and sign in or create an account.
  2. Open the API Keys page.
  3. Click Create key, give it a name such as ExClaw, and copy the key — it starts with sk-ant-.
  4. Add billing information. Anthropic charges pay-as-you-go; there is no monthly subscription required.
Cost in practice. Asking ExClaw to describe a workbook is cheap — a few cents per session is typical. Long back-and-forth conversations use more tokens; switch to Haiku for routine work and Sonnet for anything analytical.

Chapter 3 — Your first questions

Chapter 3Your first questions

Open any workbook — ideally one you did not create yourself — and click ExClaw on the Home ribbon. By the end of this chapter you will have seen ExClaw describe a workbook, enumerate its worksheets, and answer follow-up questions against the live file.

3.1 The chat panel

  • Header — product name and a gear icon for settings.
  • Conversation — your messages on the right, ExClaw's on the left.
  • Status bar — appears while ExClaw is working. Progress bar, queued tool calls and a Cancel button.
  • Input box — Ctrl+Enter or Send submits; plain Enter adds a new line.

3.2 A guided example

Try these in order.

Overview

Tell me what this workbook contains. How many worksheets, roughly how much data in each, and which ones look like inputs vs. outputs?

ExClaw calls get_workbook_info and list_worksheets, then answers in plain English.

Pick a sheet to look at first

Based on what you just told me, which worksheet should I open first to understand the main model?

Cross-check

Is there a sheet that looks like a summary or dashboard? If so, which one?

3.3 How ExClaw thinks

Every instruction goes to Claude along with a short description of your Excel context (which workbook is active, which worksheet is selected). Claude breaks the request into one or more tool calls — for now that's get_workbook_info and list_worksheets — which ExClaw runs against the live workbook. The result feeds back into Claude, and it decides whether it needs more data before answering.

Things to know:

  • ExClaw can only see what its tools expose. Today that is workbook metadata and worksheet structure — not cell values.
  • If you ask about specific numbers, ExClaw will tell you it does not currently have a way to read cells. That is a known limitation of this release.
  • Nothing you do in the chat panel changes the workbook. Close without saving and nothing is lost.
Don't expect formula audit yet. ExClaw cannot read cell contents, inspect formulas or trace precedents in this release. For those tasks, Excel's built-in Formula Auditing and Power Query are still your best tools.

Chapter 4 — Workbook & worksheets

Chapter 4Workbook & worksheets

This chapter covers the workbook-level tools — the ones that describe and orient you inside the file you have open. Per-cell and per-feature tools are in the next chapter.

4.1 get_workbook_info

Returns metadata about the active workbook — the one currently in focus in Excel. No arguments.

FieldWhat it tells you
WorkbookThe file name, e.g. Q2-forecast.xlsx.
Full pathThe absolute path on disk. Lets you tell two similarly-named files apart.
SavedTrue if every change has been saved; False if there are unsaved edits.
Sheet countTotal number of sheets (including chart sheets, hidden sheets, etc.).
Active sheetThe sheet currently selected. Worksheet-only — chart sheets are not reported here.
Calc modeAutomatic, Manual or Semi-automatic. Worth checking before you interpret any numbers.
What workbook do I have open, and is it in manual or automatic calc mode?

4.2 list_worksheets

Enumerates every worksheet in the active workbook, with its used-range bounds. No arguments.

FieldWhat it tells you
index1-based position of the sheet in the tab order.
nameSheet name (the label on the tab).
used_rowsHow many rows the used range spans. A good proxy for "how much is in this sheet".
used_colsHow many columns the used range spans.
List every worksheet with its size. Flag any that are suspiciously empty or suspiciously large.
Used range is not the same as data range. Excel's used range is defined by the last cell that has ever been edited, not the last cell with content today. A sheet that once had 10 000 rows and was cleared may still report 10 000 used rows. Treat the numbers as an upper bound.

4.3 File-level tools

Alongside the inspection tools, ExClaw supports the same file-level operations you would reach through the File menu:

  • Create a new workbook. "Create a new blank workbook." Opens a fresh Excel window next to the current one. Add a save path for immediate save.
  • Start from a template. "Create a new workbook from C:\Templates\Budget.xltx." Excel opens the template as an unsaved copy — the template file itself is not modified.
  • Open an existing workbook. "Open C:\Reports\Q2.xlsx." The file opens in a new window; your chat pane continues driving the workbook it started in.
  • Inspect a template before you use it. "What's in C:\Templates\Budget.xltx?" — get_template_file_info opens the template silently, reports its sheets, their used-range dimensions, visibility, named ranges, and defined tables, then closes it. get_template_info gives a lighter summary when you just need sheet names.
Chat pane stays put. When ExClaw creates or opens another workbook, the new window opens alongside yours but the chat pane remains bound to the workbook it started in. Open a second chat pane in the new window if you want the AI driving both.

4.4 Beyond the workbook overview

The three sections above cover the workbook-level tools. ExClaw can also reach into the cells, tables, charts, shapes and formatting inside your workbook — Chapter 5 walks through those capabilities organised by task.

4.5 Example recipes

Workbook handover

I just opened a workbook someone else built. Describe it so I know what to look at first — file name, sheet count, which sheets are big, which look like summaries.

Sanity check before a meeting

Is this workbook saved, and is calc mode set to automatic? If not, tell me — I'm about to screen-share.

Finding the dashboard

Chapter 5 — Inside the workbook

Chapter 5Inside the workbook

Chapter 4 covered the workbook as a whole. This chapter walks through everything ExClaw can do inside a workbook — ranges and formulas, tables and pivot tables, charts and shapes, formatting, and the review / export surface. Each section gives a short list of what the AI can reach and a few example prompts you can paste into the chat panel.

5.1 Reading and writing ranges

ExClaw reads any range you point it at and writes values, formulas and array formulas back. It can also clear ranges, fill series, and import or export CSV data.

  • Read a range. "Read A1:E20 on Sheet1." ExClaw returns the cell values in a table you can copy or summarise.
  • Write values or formulas. "Put the heading 'Revenue' in B1 and =SUM(B2:B100) in B101." Plain text, literal formulas, or full array formulas are all accepted.
  • Clear or fill. "Clear the contents of C5:C25" / "Fill A2:A13 with January through December."
  • Copy ranges. "Copy the totals block from Summary!A1:C10 to the same position on Sheet2."
  • Import / export CSV. "Import C:\Data\orders.csv into a new sheet called Orders" / "Export the Orders table to C:\Exports\orders.csv."
  • Find and replace. "Find every occurrence of 'Q1' and change it to 'Q1 2026'."
  • Trace dependencies. "Where does D45 get its value from?" — ExClaw walks precedent and dependent cells.

5.2 Sheets

Full sheet lifecycle: add, delete, rename, duplicate, move, colour tabs, hide, freeze, protect.

  • Add, rename, delete, duplicate, move. "Add a sheet called Forecast at the end" / "Delete the Drafts sheet" / "Move Summary to be the first sheet."
  • Tab colour and visibility. "Colour the Archive tab grey and hide it."
  • Freeze and view. "Freeze the first row and the first two columns on every sheet" / "Hide gridlines on Dashboard."
  • Protect. "Protect the Locked_Template sheet but allow column formatting" — ExClaw applies sheet protection with the right flags.

5.3 Tables, named ranges and pivot tables

ExClaw treats structured tables as first-class objects — they are easier for the AI to reason about than bare ranges.

  • Create a table. "Convert A1:F500 on Sales into a table called Sales2026." ExClaw calls create_table, enables the header row, and applies a clean style.
  • Totals, filter, sort. "Add a totals row to Sales2026 that sums the Revenue column" / "Filter Sales2026 to only show rows where Region='North'" / "Sort the table descending by Revenue."
  • Named ranges. "Name B2:B100 on Sheet1 as Revenue2026" — afterwards you can refer to it by name in any formula.
  • Pivot tables. "Build a pivot table from Sales2026 with Region in the rows, Category in the columns, and Revenue as the values." ExClaw creates the pivot, adds fields, and sets filters.
  • Refresh. "Refresh all pivots and external connections."

5.4 Charts, shapes and visuals

ExClaw inserts and modifies charts, shapes, pictures and hyperlinks.

  • Insert a chart. "Add a column chart of Sales2026 showing Revenue by Region, placed on Dashboard starting at B20, 500×300 pixels."
  • Modify an existing chart. "Switch the chart on Dashboard to a stacked bar" / "Add a second series for Cost" / "Show data labels on every point."
  • Shapes and pictures. "Insert a rounded-rectangle callout on A1 saying 'Q1 peak'" / "Embed the logo at C:\Brand\logo.png on every sheet's A1 cell."
  • Hyperlinks and comments. "Add a hyperlink on A1 that jumps to the Summary sheet" / "Leave a comment on D45 saying 'verify with finance'."

5.5 Formatting

ExClaw applies number formats, cell styles, borders, colours and conditional formats.

  • Number formats. "Format B:B as currency with two decimals" / use named presets: "accounting", "percent", "date-iso", etc.
  • Cell formatting. "Make the header row bold, centred, with white text on a dark-blue fill" — one call via apply_cell_format.
  • Borders. "Add a thin border around every cell in A1:F500."
  • Named styles. "Apply the Heading 2 style to every sheet's A1." Built-in styles and user-defined styles both work.
  • Conditional formats. "Highlight any cell in the Revenue column that is below its column average in red." ExClaw composes the rule; use clear_conditional_formats to remove them later.
  • Data validation. "Add a drop-down on E2:E100 with the values North, South, East, West."

5.6 Review, export and sharing

  • Export PDF. "Export Dashboard and Summary to C:\Reports\Q1.pdf as a single file."
  • Page setup. "On Summary, set the print area to A1:F60, fit to one page wide, landscape."
  • Repeat rows / column breaks. "Repeat row 1 on every printed page" / "Insert a manual page break after row 50."
  • Comments and revisions. ExClaw can list all comments, add new ones, and clear them.
  • Protection. Sheet protection, workbook structure protection, and per-range lock state are all scriptable.

5.7 Advanced: macros, calc mode and connections

  • Run a macro. "Run the macro ReformatInvoice on Sheet1" — ExClaw invokes run_macro with any arguments you supply. Use list_macros to discover what is available.
  • Calculation mode. "Switch calculation to manual while I make these changes" / "Recalculate the whole workbook now."
  • External connections. "Refresh the data connections and the pivots" / refresh a specific query by name.

5.8 Example recipes

Build a mini dashboard

On a new sheet called Dashboard, make a column chart of Sales2026 showing Revenue by Region, a line chart of monthly Revenue, and three callouts with the total revenue, total cost, and gross margin. Use the heading style for the sheet title.

Clean up an inherited workbook

Audit this workbook. List every hidden sheet, every range with conditional formatting, and every defined name that points nowhere. Then suggest which ones are safe to delete.

Turn a raw CSV into a reportable table

Import C:\Data\orders-jan.csv into a new sheet, convert the imported range into a table called OrdersJan with the Accounting number format on the amount column, add a totals row, and freeze the header.

Chapter 6 — Settings

Chapter 6Settings

ExClaw's settings window is reached from the gear icon in the chat panel header. Fields are deliberately minimal.

6.1 API key

Your Anthropic API key, encrypted with Windows DPAPI and stored under %APPDATA%\ExClaw\. Click Show to reveal it temporarily.

Rotate your key if you share a machine. DPAPI protects keys from other Windows accounts on the same machine, but if you allow another person to use your Windows session, they can use your API key. Generate a new key in the Anthropic console and paste it here.

6.2 Model

Chooses which Claude model answers your messages.

ModelWhen to use
HaikuFast and cheap. Good for routine tasks with small inputs and short replies.
SonnetBalanced default. Handles summarisation, multi-step reasoning and longer drafts.
OpusHighest quality, slower and more expensive. Use for nuanced tone, sensitive material or hard judgement calls.

6.3 Logging

Controls the local log file used for diagnostics.

  • Off — no log written.
  • Errors only (default) — only failures are recorded.
  • Verbose — every tool call, input and result is written to the log.

Log file location: %APPDATA%\ExClaw\exclaw.log. It is a plain text file you can open in Notepad. Document, mailbox or workbook contents are not logged at any verbosity level.

6.4 Panel position

By default the chat panel docks on the right of the main Excel window. You can float it or dock it on the left. The position is remembered per Windows profile.

Chapter 7 — Tips & troubleshooting

Chapter 7Tips & troubleshooting

7.1 Getting the most out of ExClaw

  • Describe the outcome, not the formula. "Add a totals row to the orders table" beats hand-crafting a SUBTOTAL(...) — the AI picks the right formula and places it on the right row.
  • Name your ranges and tables. ExClaw is far more reliable when you say "the Quarterly_Revenue table" than when you say "the one on sheet 3 that starts around B4".
  • Undo fearlessly. Every ExClaw change is a normal Excel edit, so Ctrl+Z works. Try something; if you do not like it, undo.
  • Clear the conversation periodically. Resets the context and reduces token usage on subsequent turns.

7.2 Common pitfalls

SymptomCauseFix
ExClaw says "no active workbook"Excel is open but has no workbook in focus (e.g. Backstage view is showing).Click into any workbook window and retry.
Sheet count seems too highHidden sheets and chart sheets are included in the count.In Excel, right-click a tab and choose Unhide to inspect hidden sheets.
Used-range values look hugeExcel tracks the largest ever used cell, not the current data.In Excel, press Ctrl+End to see what Excel thinks the last cell is. Clear the phantom area and save to reset.
Answers ignore a sheetVery hidden sheets (set via VBA) are not enumerated by the Sheets collection.Unhide via VBA (xlSheetVisible) if you need ExClaw to see them.

7.3 Cost and token tips

  • Workbook descriptions are cheap — the tool results are small. Most cost is the reply itself.
  • Haiku is usually enough for this release. Reach for Sonnet when you want pattern-spotting across many sheets.

7.4 Bug-report helpers

Two built-in tools capture the state support needs:

  • get_version_info — ask "what version of ExClaw am I running?" and the reply includes add-in name, auto-refreshed build version, the Excel version, OS, and .NET runtime. Same version is always visible in the top-right corner of the Settings dialog.
  • list_tools — ask "which ExClaw tools mention 'pivot'?" and the agent filters the full tool catalogue by any substring. Handy when you aren't sure whether a capability exists for your scenario.

7.5 Where to get help

  • Log file — %APPDATA%\ExClaw\exclaw.log. Set logging to Verbose before reproducing the issue.
  • kalmstrom.com support — send the log excerpt (with your API key redacted) and a short description of what you asked and what happened.
  • Feature requests — tell kalmstrom.com which Excel capabilities matter most for your workflow. Future releases will be driven by real user requests.
